Python で生成した CSV を Excel に取り込み、ガントチャート上に進捗線を描画する処理フェーズを 
簡易化した VBA サンプルコード です。

📌 Overview
本リポジトリは、業務システムで使用している
「Python → Excel → VBA」連携パイプラインの最終フェーズ を
転職用に安全な形で再構成したものです。

実務では以下のような処理を VBA で自動化しています：

・Python で前処理された CSV の Excel 取り込み
・ガントチャート上の作業行・日付列の自動特定
・進捗率に応じた 進捗線（Zigzag Line） の描画
・不要線の判定・削除
・描画履歴のログ出力
・Shapes API を用いた動的な線分生成

ただし、業務ロジックは機密のため公開できないため、
本リポジトリでは 安全な範囲のサンプルコードのみ を掲載しています。
